# Example prompts — MAX docs integration

## Prompt 1 — Endpoint lookup
```text
Using the MAX docs MCP server, show me which endpoint I should use to send a message and explain the required fields.
```

## Prompt 2 — Guide retrieval
```text
Use the MAX docs server to find the guide for bot creation and summarize the setup steps.
```

## Prompt 3 — Bridge API grounding
```text
Check the MAX Bridge API docs and list the available events, then tell me how a mini app would react to the back button.
```

## Prompt 4 — UI component lookup
```text
Find the MAX UI Button component in the docs and summarize the main props and intended use.
```
