# launchery-mcp-examples v1.0.0

Runnable MCP examples that connect the Launchery portfolio into one practical learning path.

## Highlights
- MAX docs integration walkthrough
- Custom MCP server walkthrough using `mcp-starter-typescript`
- MCP health-check workflow for validation and debugging
- blueprint docs for Multi-MCP setup and MCP + LLM integration

## Included examples
1. **MAX docs integration**
   - build `max_docs_mcp`
   - wire it into an MCP client config
   - run concrete prompts against grounded docs

2. **Custom MCP server**
   - start from `mcp-starter-typescript`
   - add a focused custom tool
   - build and test locally

3. **MCP health-check**
   - validate a server before client integration
   - see pass/fail output patterns
   - use troubleshooting commands to fix common issues

## Why this release matters
This repo turns the Launchery MCP portfolio into a connected learning surface instead of isolated repositories.
